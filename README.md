# Telegram-бот «Список дел» с напоминаниями

Бот для управления личными задачами прямо в Telegram. Написан на **aiogram 3**
(современный асинхронный фреймворк для Telegram-ботов).

## Возможности

- `/add купить хлеб` — добавить задачу
- `/list` — список задач с отметками ✅/🔄
- `/done 2` — отметить задачу №2 выполненной
- `/delete 2` — удалить задачу №2
- `/remind 30 позвонить маме` — напоминание через 30 минут (работает в фоне,
  не блокирует другие команды)

## Технологии

- Python 3.10+, asyncio
- aiogram 3.x — обработка команд через фильтры и роутинг
- Фоновые задачи через `asyncio.create_task` для напоминаний

## Запуск

```bash
pip install -r requirements.txt
```

1. Напишите [@BotFather](https://t.me/BotFather) в Telegram → `/newbot` → получите токен.
2. Создайте файл .env и пропишите в него токен ```BOT_TOKEN=8913:AFJKL``` (токен не хранится в коде — это безопасно):

```bash
python bot.py
```

## Что демонстрирует проект

- асинхронная архитектура: бот обрабатывает много пользователей одновременно;
- валидация пользовательского ввода (номер задачи, лимиты времени напоминания);
- разделение задач по пользователям;
- безопасная работа с токеном через переменные окружения;
- фоновые задачи без блокировки основного цикла.
